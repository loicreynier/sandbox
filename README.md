# Sandbox

Sandbox for random tests that I may need in the future.

## Sand castles

- `detect-windarkmode`: dirty function to detect Windows dark mode
- `gmres`: dirty unoptimized Python implementation of the GMRES algorithm
- `gmres-sp`: wrapper for SciPy's GMRES
- `nmlinput`: Fortran helper routines to use namelists as input files
- `regula-falsi`: unoptimized C implementation of the Regula falsi method
